This folder contains files downloaded from the CisBP-RNA database (http://cisbp-rna.ccbr.utoronto.ca/) for the set of RBPs in the RNAcompete paper (Ray et. al. 2013). The contents of each file is a position weight matrix (pwm). Each file is named with the name of the protein followed by the organism of origin. Some proteins have multiple associated pwms, indicated by a number at the end of the file name.
